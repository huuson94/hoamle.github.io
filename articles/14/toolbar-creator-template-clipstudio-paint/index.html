Toolbar-Creator template for Photoshop and ClipStudio Paint

October 31, 2014

for Photoshop (R) and ClipStudio Paint aka. Manga Studio (R)


on GitHub

Toolbar Creator is one of few especially useful on-screen toolbars for Windows Tablet PCs. It makes using Windows tablets without keyboard much more comfortable and convenient. The software is developed by lblb at TabletPC Review forum, which is a user-friendly interface for RawInputControlTest script by koide.

This repository is a toolbar template, specifically designed for painting in Adobe (R) Photoshop (R) and ClipStudio Paint aka. Manga Studio (R). Icon design is kindly provided by gahfe at TabletPC Review forum. The template was created, assuming that modifier keys (Ctrl, Alt, Shift) reside on an external harware device, eg. keyboard, tablet buttons, gamepad etc. I have mine on a Zeemote controller.

Installation

  1. Download and extract this repo
    • (for non-GitHub users): click here, extract the downloaded file into a folder, and go into Toolbar-Creator-master folder.
    • (for GitHub users): simply clone with git
  2. Copy and paste folder [version]\Files into your Toolbar Creator folder. Replace [version] with either v2_1 or v2_2_b5 accordingly to your existing Toolbar Creator version.
  3. Replace/Merge all existing files/folders if prompted.
  4. Launch Toolbar Creator and change the current toolbar to painting.
